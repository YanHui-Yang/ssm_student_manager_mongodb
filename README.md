### 基于Spring + Spring MVC + Spring Data MongoDB写的学生管理系统，前端是Bootstrap，数据库采用MongoDB，实现增删改



### 运行方式

在mongo执行

```sql
use student_manager

db.user.insert({"username": "admin", "password": "123456"})
```
